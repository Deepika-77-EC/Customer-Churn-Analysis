## Project Overview
This project focuses on analyzing customer churn in a subscription-based business to understand why customers leave and how retention can be improved.

## Tools & Technologies
- SQL (MySQL)
- Python (Pandas, Matplotlib, Seaborn)
- Excel (Pivot Tables, Charts)

## Key Objectives
- Identify churn patterns using SQL queries
- Clean and analyze customer data using Python
- Visualize churn trends
- Create Excel dashboards for business reporting
- Suggest data-driven retention strategies

## Key Insights
- Customers on monthly contracts have higher churn rates
- Higher monthly charges correlate with increased churn
- Long-term plans can significantly improve retention

## Business Recommendations
- Promote yearly and long-term subscription plans
- Offer discounts to high-risk customer segments
- Improve customer engagement for high-value users
